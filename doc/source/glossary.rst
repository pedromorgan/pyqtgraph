======================
Glossary
======================

.. glossary::

    NumPy
        NumPy is the fundamental package for scientific computing with Python
            
        * http://www.numpy.org/
        
    Python
        The Python Interpereter
        
        * http://python.org
        
        
    PyQt
        PyQt are python binding to the :term:`Qt` toolkit
        
        * http://www.riverbankcomputing.co.uk/software/pyqt/i
        
    PySide
        PySide are python bindings to the :term:`Qt`
        
        * http://qt-project.org/wiki/PySide
        
        
    
    Qt
        Qt (pronounced “Cute”) is a cross-platform application and UI framework, developed using C++
        
        * http://qt-project.org/
        
        
    ROI
        Region of interest
        
        
    SciPy
        SciPy (pronounced “Sigh Pie”) is a Python-based ecosystem of open-source software for mathematics, science, and engineering.
        
        * http://www.scipy.org/
